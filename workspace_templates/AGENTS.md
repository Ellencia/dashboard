# 워크스페이스 작업 지침 — Codex (AGENTS.md)

> 이 파일은 OpenAI Codex CLI가 자동으로 읽음.
> 같은 내용이 CLAUDE.md(Claude Code), GEMINI.md(Gemini CLI)에도 있음.

## STATUS.md / update.md 유지 (필수)

이 워크스페이스(이 파일이 있는 작업 폴더)의 각 프로젝트 폴더 최상위에는
**STATUS.md**(진행 상황)와 **update.md**(변경 이력) 두 파일을 둔다.
여기서 코드 작업을 할 때는 항상 다음을 지킨다:

1. 작업 시작 전 이 폴더 최상위의 **`01_code.md`** 를 읽고 그 양식을 따른다.
   (01_code.md = 두 파일 작성 규격의 단일 출처)
2. 작업으로 진행 상황이 바뀌면 해당 프로젝트 **STATUS.md** 를 갱신한다.
   끝낸 항목은 `- [x]`, 새 작업은 `- [ ]` 로.
3. 작업을 했으면 해당 프로젝트 **update.md** 맨 위에 오늘 날짜
   (`## YYYY-MM-DD`) 블록으로 변경 내용을 한 줄 이상 기록한다.
4. 두 파일은 **UTF-8** 로 저장하고, 체크박스·날짜 형식을 01_code.md 대로 정확히 지킨다.

상세 규격·예시·주의사항은 `01_code.md` 를 참조할 것.
