# 여기에 프로젝트 표시 이름

> 한 줄 메모 (선택 사항, 위젯에 작게 표시됨)

- [x] 이미 끝낸 작업
- [x] 또 다른 끝낸 작업
- [ ] 아직 남은 할 일
- [ ] 또 다른 할 일

<!--
사용법
  1) 이 파일을 추적하고 싶은 프로젝트 폴더에 STATUS.md 라는 이름으로 복사
  2) 맨 윗줄 "# 제목" 을 프로젝트 이름으로 수정
  3) "- [ ]" 는 미완료, "- [x]" 는 완료 항목
  4) 진행률(%) = 체크된 항목 수 / 전체 항목 수
  5) 미완료 항목들이 위젯의 "할 일" 목록에 모임

규칙
  - 맨 위 첫 "# 제목" 한 줄이 표시 이름이 됨 (없으면 폴더 이름 사용)
  - 첫 ">" 인용 줄 한 개가 메모로 표시됨
  - 체크박스 줄은 반드시 "- [ ]" 또는 "- [x]" 로 시작 ("[]" 안 됨, 대괄호 안 공백 필수)
  - 이 주석 블록은 위젯에 표시되지 않으므로 지워도 됨
-->
